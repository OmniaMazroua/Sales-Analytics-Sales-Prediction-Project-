# Sales Analytics & Sales Prediction Project

## Project Overview

This project analyzes sales data and builds machine learning models to predict sales amounts. The workflow includes data cleaning, exploratory data analysis (EDA), feature engineering, visualization, and model evaluation.

## Objectives

* Analyze sales performance and profitability.
* Identify trends and patterns in sales data.
* Build predictive models to estimate sales amounts.
* Compare the performance of multiple machine learning algorithms.

## Dataset

The dataset contains sales transactions with features such as:

* Sale Date
* Product Category
* Quantity Sold
* Unit Price
* Unit Cost
* Discount
* Sales Amount

## Data Preparation

* Converted date fields into proper datetime format.
* Created a new feature: **Profit**.
* Removed unnecessary columns.
* Checked for missing values and outliers.
* Prepared data for machine learning models.

## Exploratory Data Analysis (EDA)

Key analyses performed:

* Sales distribution
* Profit analysis
* Product category performance
* Monthly sales trends
* Outlier detection using IQR
* Sales and profit visualizations

## Machine Learning Models

The following regression models were trained and evaluated:

1. Linear Regression
2. Decision Tree Regressor
3. Random Forest Regressor
4. Gradient Boosting Regressor
5. XGBoost Regressor

## Evaluation Metrics

Models were compared using:

* R² Score
* RMSE (Root Mean Squared Error)
